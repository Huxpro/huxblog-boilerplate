## pycocotools

mmdetecton3d 运行报错：``pycocotools.__version__ >= '12.0.2``

通过参考[本文](https://github.com/open-mmlab/mmdetection/issues/3761)解决。
```
conda activate mm
pip uninstall pycocotools
pip install mmpycocotools
pip uninstall mmpycocotools
pip install mmpycocotools
```
