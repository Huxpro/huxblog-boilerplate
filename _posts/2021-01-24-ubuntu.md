## nohup
后台运行
```
nohup bash ./train.sh >> out.log 2>&1 &
```


## killall
停止某一用户的全部进程
```
killall -u username
```

## scp
文件传输
```
scp -P 15654 zhouzheyuan@10.192.9.10:/var/lib/docker/data/datasets/coco/test2017.zip ./
```


## du
显示文件夹大小
```
du -sh
```
